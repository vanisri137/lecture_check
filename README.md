# Lecture Check

Lecture Check is a web-based application developed to analyze and manage lecture-related content efficiently. The project includes functionalities for uploading files, transcription handling, and similarity checking using Python-based processing.

## Features

- Upload lecture/audio files
- Store and manage uploaded content
- Transcription processing
- Similarity checking between lecture contents
- Frontend interface built with React
- Backend integration using Node.js

## Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Additional Tools
- Python (Similarity Checking)
